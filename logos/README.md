# Enaga Logo Concepts

25 logo designs for the Enaga white noise app, inspired by the Shima Enaga bird (*Aegithalos caudatus trivirgatus*) from Hokkaido, Japan.

**Design language:** White rounded-square icon. Just two eyes and a beak — nothing else. Clean and minimal.

All logos are SVG (vector) files: scalable to any size, minimal file size.
