# Enaga Logo Concepts

25 logo designs inspired by the Shima Enaga bird (*Aegithalos caudatus trivirgatus*) — a tiny, round, fluffy long-tailed tit native to Hokkaido, Japan. The bird is famous for its almost completely white face and round, cloud-like shape.

All logos are SVG (vector) files: scalable to any size, minimal file size.

## Bird Reference

The **Shima Enaga** (シマエナガ) is known for:
- Nearly all-white face and body
- Tiny dark eyes
- Dark forehead/cap
- Round, fluffy appearance
- Long tail (not shown in face-only logos)
